Sobre o Projeto

Este projeto foi desenvolvido para aprender os conceitos básicos de Spring Boot, utilizando Thymeleaf para renderização de páginas web.

A proposta é simular um pequeno sistema de catálogo de filmes, onde é possível navegar entre páginas e, futuramente, salvar dados no banco PostgreSQL.

Tecnologias utilizadas:

  • Java
  • Spring Boot
  • Thymeleaf
  • HTML, CSS e JavaScript
